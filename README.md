# loco-lab

LocoLab is an umbrella project for the Loco research lab, focusing on a wide range of topics in the field of artificial intelligence, including applied AI, conversational AI, education technology, and privacy-focused local AI solutions.

## Installation

To set up the LocoLab project, follow these steps:

```bash
git clone https://github.com/michael-borck/loco-lab.git
cd loco-lab
npm install
```

## Usage

The LocoLab project consists of several components, each with its own usage instructions:

### CLI Tool

The LocoLab project includes a command-line interface (CLI) tool for interacting with the project's features. To use the CLI tool, run the following command:

```bash
npm run cli
```

This will start the CLI tool and allow you to explore the available commands and functionalities.

### Conversational AI

The LocoLab project includes a conversational AI component, which allows users to engage in natural language interactions. To use the conversational AI, run the following command:

```bash
npm run start:conversational-ai
```

This will start the conversational AI server, and you can interact with it using a compatible client.

### Education Technology

The LocoLab project includes an education technology component, which provides tools and resources for educators and learners. To access the education technology features, visit the project's documentation website:

```
https://loco-lab.github.io/
```
